## 🔧 التحقق السريع

### في المتصفح
```
1. افتح الموقع
2. اضغط F12 (أدوات المطور)
3. انتقل إلى Lighthouse
4. شغّل الاختبار
5. تحقق من النتائج > 90
```

### على الموبايل
```
1. افتح على هاتفك
2. تحقق من الألوان
3. جرّب الأزرار
4. اختبر القائمة
5. تحقق من السرعة
```

### لوحة المفاتيح
```
1. اضغط Tab = تنقل
2. اضغط Enter = اضغط
3. اضغط Escape = إغلاق القائمة
4. اضغط Arrow = في carousel
```
